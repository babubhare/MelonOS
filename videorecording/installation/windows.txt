1. Install MSYS2 (https://www.msys2.org/)
2. Open the mingw64.exe (C:\msys64)
3. Update Packages (pacman -Syu)
4. Install gcc (pacman -S gcc)
    a. https://packages.msys2.org/package/gcc?repo=msys&variant=x86_64
5. Install nasm (pacman -S nasm)
    a. https://packages.msys2.org/package/nasm?repo=msys&variant=x86_64
6. Install mingw-w64-x86_64-qemu (pacman -S mingw-w64-x86_64-qemu)
    a. https://packages.msys2.org/package/mingw-w64-x86_64-qemu?repo=mingw64
    b. https://www.qemu.org/download/#windows
7. Install git (pacman -S git)
    a. pacman -S git
